# Obsidian templates

Snippets used by the Obsidian core "Templates" plugin. Trigger via
`Ctrl/Cmd+P` → "Insert template".

These are different from the YAML/Markdown skeletons in the parent
`_templates/` folder — those are full file/folder layouts you copy
manually. These are inline snippets you insert into an open file.

## Available

| Template | Inserts |
|----------|---------|
| `lesson-frontmatter.md` | YAML frontmatter for a new lesson |
| `question-multiple-choice.md` | One MC question entry for `questions.yaml` |
| `question-true-false.md` | One T/F question entry |
| `question-free-text.md` | One free-text question entry |
| `question-coding.md` | One coding question entry |
| `directive-card.md` | `> [!card] slug` callout |
| `directive-challenge.md` | `> [!challenge] slug` callout |
| `directive-callouts.md` | All native Obsidian callout types |

All callouts use Obsidian's native syntax — what you see in Obsidian's
preview is exactly what students see on the platform.
